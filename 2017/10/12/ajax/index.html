AJAX编程

撰写于 2017-10-12 修改于 2017-10-13

AJAX(Asynchornous Javascript And XML) AJAX不是一门新的语言，而是对现有的技术的综合利用。本质上是在HTTP协议的基础上以异步的方式与服务器进行通信。

异步

指某段程序执行时不会阻塞其他程序的执行，其表现形式为程序的执行顺序不依赖书写顺序，相反则为同步。其优势在于不阻塞程序的执行，提高执行效率。XMLHttpRequest可以以异步的方式出路程序。

XMLHttpRequest

浏览器内置对象，用于在后台与服务器通信(交换数据)，由此我们便可以实现对网页的部分刷新，而不是刷新整个页面。

响应

HTTP响应是由服务端发出，作为客户端更应关心的是响应的结果
HTTP响应由3个组成部分，与XMLHttpRequest方法或属性的对应
由于服务器做出响应需要时间(比如网速慢等原因)，所以我们需要监听服务器的响应状态，然后才能进行处理。

readyState
0: 请求未初始化，还没有调用open();
1: 请求已建立，但是还没发送(还没调用send());
2: 请求已发送，正在处理中(通常这个时候可以从响应头中获取内容)
3: 请求在处理，通常响应中已有部分数据可用了，但是服务器还没完成响应;
4: 响应已完成，可以获取并使用服务器的响应。

onreadysatechange是Javascript的事件的一种，其意义在于监听XMLHttpRequest的状态

  1. 获取响应状态行(包括状态码&状态信息)

    xhr.status / 状态码 /
    xhr.statusText / 状态信息 /

  2. 获取响应头

    xhr.getResponseHeader(‘Content-Type’);
    xhr.getAllResponseHeaser();

  3. 响应主体

    xhr.responeText
    xhr.responseXML

我们需要监听并判断响应头的MIME类型后确定使用requset.responseText或者request.responseXML

API详解

xhr.open() 发起请求，可以是get、post方式
xhr.setRequestHeader() 设置请求头
xhr.send() 发送请求主体， get 方式使用xhr.send(null)
xhr.onreadstatechange = function () { } 监听响应状态
xhr.status 响应码，如200
xhr.statusText 表示响应信息，如OK
xhr.getAllResponseHeaders() 获取全部响应头部信息
xhr.getResponseHeader(‘key’) 获取指定头信息
xhr.responseText、xhr.responseXML 都表示响应主体

注：GET和POST请求方式的差异

  1. GET没有请求主体，使用xhr.send(null)
  2. GET可以通过在请求url上添加请求参数
  3. POST可以通过xhr.send(‘name=it&age=12’); 设置请求数据
  4. POST需要设置请求头 xhr.setRequestHeader(‘Content-Type’,’application/x-www-form-urlencode’);
  5. GET效率更好（应用多）
  6. GET大小限制约4k，POST没有限制

如何获取复杂数据呢？

XML

语法：

  1. 必须有一个根元素
  2. 不可有空格，不可以数字或 . 开头，大小写敏感
  3. 不可交叉嵌套
  4. 属性双引号
  5. 特殊符号要用实体
  6. 注释和HTML的一样

虽然可以描述和传输复杂数据，但是其解析过于复杂并且体积较大，所以实现开发已经很少用了

JSON

即JavaScript Objeck Notation，另一种轻量级的文本数据交换格式，独立于语言

语法:

  1. 数据在名称/值中
  2. 数据有逗号分隔开
  3. 中括号保存数组，花括号保存对象
  4. 属性使用双引号

JSON数据在传输的过程中是以string形式，不同的语言都有各自的解析方式，需要解析之后才可以读取

PHP解析方法

将php数组转化成json字符 json_encode()
将json字符转化成php数组 json_decode()

JavaScript解析方法

利用JSON对象，把JSON对象转化成字符串：JSON.stringify();
把字符串转化成JSON对象：JSON.parse()

JSON兼容处理json2.js
总结：JSON体积小、解析方便且高效，在实际开发成为首选。

封装AJAX工具

jquery的ajax

jquery为我们提供了更强大的AJAX封装

  1. $.ajax({}) 可配置方式发起Ajax请求
  2. $.get() 以GET方式发起Ajax请求
  3. $.post() 以POST方式发起Ajax请求
  4. $(‘form’).serialize() 序列化表单（即格式化key=val&key=val）
  5. url 接口地址
  6. type 请求方式
  7. timeout 请求超时
  8. dataType 服务器返回格式
  9. data 发送请求数据
  10. beforeSend: function () {} 请求发起前调用
  11. success 成功响应后调用
  12. error 错误响应时调用
  13. complete 响应完成时调用（包括成功和失败）
